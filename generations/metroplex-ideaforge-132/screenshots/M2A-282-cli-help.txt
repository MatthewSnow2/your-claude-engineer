Usage: cli [OPTIONS] COMMAND [ARGS]...

  SkillHub CLI - Manage agent skills for MCP.

  A command-line tool for publishing, discovering, and managing agent skills
  based on the Agent Skills open standard.

Options:
  --version  Show the version and exit.
  --help     Show this message and exit.

Commands:
  init      Initialize a new skill directory with template files.
  install   Install a skill from the registry or local .skillpkg file.
  list      List available or installed skills.
  publish   Publish skill to MCP registry.
  search    Search for skills in the MCP registry.
  validate  Validate skill manifest against Agent Skills schema.
