FIXES APPLIED
================================================================================

[CRITICAL] Fix #1: Path Traversal Vulnerability
------------------------------------------------
File: src/skillhub/core/cache.py
Issue: File paths from files dictionary used directly without validation
Fix: Added path traversal protection using .resolve() and .is_relative_to()
     Validates all paths are within skill_version_dir before writing

[HIGH] Fix #2: Missing File Size Validation  
---------------------------------------------
File: src/skillhub/core/cache.py
Issue: Files written without size checks
Fix: Import MAX_FILE_SIZE (10MB) and MAX_PACKAGE_SIZE (100MB)
     Validate individual file sizes and total package size
     Reject oversized packages with clear error messages

[HIGH] Fix #3: Version String Injection Risk
----------------------------------------------
Files: src/skillhub/core/cache.py, src/skillhub/core/registry.py
Issue: Version sorting crashes on malformed version strings  
Fix: Wrapped version parsing in try-except blocks
     Returns [0, 0, 0] for invalid versions
     Applied consistently in both files

[MEDIUM] Fix #4: No Validation of Package Spec Input
------------------------------------------------------
File: src/skillhub/cli/commands/install.py
Issue: parse_package_spec() accepts empty/malformed input
Fix: Validate spec is not empty
     Validate @ format has both name and version
     Strip whitespace from parsed values

[MEDIUM] Fix #5: Symlink Not Filtered (install.py)
----------------------------------------------------
File: src/skillhub/cli/commands/install.py
Issue: Symlinks not filtered when collecting files
Fix: Added "and not file_path.is_symlink()" check

[MEDIUM] Fix #6: Symlink Not Filtered (registry.py)
-----------------------------------------------------
File: src/skillhub/core/registry.py
Issue: Symlinks not filtered in _download_from_local_registry
Fix: Added "and not file_path.is_symlink()" check

[MEDIUM] Fix #7: Redundant IOError Exception
----------------------------------------------
Files: src/skillhub/cli/commands/install.py, search.py
Issue: IOError is redundant (alias of OSError in Python 3+)
Fix: Changed "except (ValueError, OSError, IOError)" to "except (ValueError, OSError)"

[MEDIUM] Fix #8: Inline shutil Imports
----------------------------------------
File: src/skillhub/core/cache.py
Issue: shutil imported inline in methods instead of top-level
Fix: Moved "import shutil" to top-level imports
